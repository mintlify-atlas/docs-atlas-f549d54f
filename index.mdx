  <h2 className="text-2xl sm:text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with ValveResourceFormat in minutes
  </p>

  <Steps>
    <Step title="Install via NuGet">
      Install the ValveResourceFormat package from NuGet:
      
      <CodeGroup>
      ```bash .NET CLI
      dotnet add package ValveResourceFormat
      ```
      ```bash Package Manager
      Install-Package ValveResourceFormat
      ```
      ```xml PackageReference
      <PackageReference Include="ValveResourceFormat" Version="*" />
      ```
      </CodeGroup>
    </Step>

    <Step title="Load a resource file">
      Create a `Resource` instance and read a Source 2 file:

      ```csharp
      using ValveResourceFormat;

      var file = "materials/example.vmat_c";
      using var resource = new Resource();
      resource.Read(file);
      ```

      The library automatically detects and parses the resource type.
    </Step>

    <Step title="Extract asset data">
      Access the parsed data and export to common formats:

      ```csharp
      using ValveResourceFormat.ResourceTypes;
      using ValveResourceFormat.IO;

      // For textures
      if (resource.DataBlock is Texture texture)
      {
          using var bitmap = texture.GenerateBitmap();
          var pngBytes = TextureExtract.ToPngImage(bitmap);
          File.WriteAllBytes("output.png", pngBytes);
      }

      // Or use the universal extractor
      using var contentFile = FileExtract.Extract(resource, null);
      File.WriteAllBytes("output", contentFile.Data.ToArray());
      ```

      <Note>
        The `FileExtract.Extract` method works with various resource types and returns the appropriate format (PNG for textures, glTF for models, etc.).
      </Note>
    </Step>
  </Steps>
</div>
