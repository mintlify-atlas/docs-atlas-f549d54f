The scene renderer provides OpenGL-based rendering for Source 2 game assets with support for physically-based rendering, shadows, post-processing, and multiple render passes.

## Renderer

Main renderer class that orchestrates scene rendering with support for shadows, post-processing, and multiple render passes.

### Methods

#### Initialize

Initializes the renderer and creates GPU resources.

```csharp
public void Initialize()
```

Must be called before rendering. Creates uniform buffers, shadow framebuffer, and compiles required shaders.

#### LoadRendererResources

Loads essential textures like BRDF lookup tables and blue noise.

```csharp
public void LoadRendererResources()
```

Loads default textures used for PBR lighting calculations. Attempts to load from game files first, falls back to embedded resources.

#### Render

Renders the scene to a framebuffer in linear color space.

```csharp
public void Render(Framebuffer framebuffer)
```

<ParamField path="framebuffer" type="Framebuffer">
  Target framebuffer with HDR color support
</ParamField>

Performs shadow pass followed by main scene rendering. Result is in linear space and requires post-processing.

#### PostprocessRender

Resolves MSAA, applies post-processing, and converts to gamma space.

```csharp
public void PostprocessRender(
    Framebuffer inputFramebuffer,
    Framebuffer outputFramebuffer,
    bool flipY = false
)
```

<ParamField path="inputFramebuffer" type="Framebuffer">
  MSAA framebuffer containing linear HDR color
</ParamField>

<ParamField path="outputFramebuffer" type="Framebuffer">
  Non-MSAA target for final LDR output
</ParamField>

<ParamField path="flipY" type="bool" default="false">
  Flip output vertically for certain rendering contexts
</ParamField>

#### Update

Updates scene state and prepares for next frame.

```csharp
public void Update(Scene.UpdateContext updateContext)
```

<ParamField path="updateContext" type="Scene.UpdateContext">
  Update context with camera, timestep, and text renderer
</ParamField>

Updates animations, recalculates matrices, sets up shadows, and collects draw calls for both main and skybox scenes.

### Usage example

```csharp
using ValveResourceFormat.Renderer;

// Create renderer context
var rendererContext = new RendererContext(fileLoader, logger);
var renderer = new Renderer(rendererContext);

// Initialize and load resources
renderer.Initialize();
renderer.LoadRendererResources();

// Set up scene and framebuffers
renderer.Scene = myScene;
renderer.MainFramebuffer = mainFramebuffer;

// Render loop
while (running)
{
    // Update scene
    var updateContext = new Scene.UpdateContext
    {
        Camera = renderer.Camera,
        TextRenderer = textRenderer,
        Timestep = deltaTime
    };
    renderer.Update(updateContext);

    // Render to HDR framebuffer
    renderer.Render(hdrFramebuffer);

    // Apply post-processing and output to screen
    renderer.PostprocessRender(hdrFramebuffer, screenFramebuffer);
}
```


#### ResolveMsaa

Resolves MSAA color and/or depth using compute shaders.

```csharp
public void ResolveMsaa(
    Framebuffer source,
    RenderTexture destColor,
    RenderTexture destDepth,
    bool resolveColor,
    bool resolveDepth
)
```

<ParamField path="source" type="Framebuffer">
  Source MSAA framebuffer
</ParamField>

<ParamField path="destColor" type="RenderTexture">
  Target for resolved color (RGBA16F)
</ParamField>

<ParamField path="destDepth" type="RenderTexture">
  Target for resolved depth (R32F)
</ParamField>

<ParamField path="resolveColor" type="bool">
  Whether to resolve color attachment
</ParamField>

<ParamField path="resolveDepth" type="bool">
  Whether to resolve depth attachment
</ParamField>

Uses Karis average for HDR-aware color resolve and min filter for depth (conservative for reverse-Z).
