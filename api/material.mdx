## Overview

The `Material` class represents a material resource containing shader parameters and texture references. It provides access to shader arguments, material attributes, and the vertex shader input signature.

**Namespace:** `ValveResourceFormat.ResourceTypes`

## Methods

### GetShaderArguments

Gets the shader arguments from integer parameters starting with "F_".

```csharp
public Dictionary<string, byte> GetShaderArguments()
```

**Returns:** `Dictionary<string, byte>` - Dictionary of shader argument names and values.

### FindD3DInputSignatureElement

Finds an input signature element by Direct3D semantic name and index.

```csharp
public static InputSignatureElement FindD3DInputSignatureElement(

```

## Accessing from Resource

Material data is accessed from the DATA block of a resource:

```csharp
using var resource = new Resource();
resource.Read("path/to/material.vmat_c");

var material = (Material)resource.DataBlock;

// Access material properties
var shaderName = material.ShaderName;
var materialName = material.Name;

// Get texture parameters
foreach (var (name, path) in material.TextureParams)
{
    Console.WriteLine($"Texture {name}: {path}");
}

// Get shader arguments (F_ parameters)
var shaderArgs = material.GetShaderArguments();
foreach (var (name, value) in shaderArgs)
{
    Console.WriteLine($"Shader Arg {name}: {value}");
}
```

## Example: Reading material parameters

```csharp
var material = (Material)resource.DataBlock;

Console.WriteLine($"Material: {material.Name}");
Console.WriteLine($"Shader: {material.ShaderName}");

// Float parameters
foreach (var (name, value) in material.FloatParams)
{
    Console.WriteLine($"  {name} = {value}");
}

// Vector parameters
foreach (var (name, value) in material.VectorParams)
{
    Console.WriteLine($"  {name} = ({value.X}, {value.Y}, {value.Z}, {value.W})");
}

// Texture references
foreach (var (name, path) in material.TextureParams)
{
    Console.WriteLine($"  {name} -> {path}");
}

// Check vertex input signature
if (material.InputSignature.Elements.Length > 0)
{
    Console.WriteLine("Vertex Inputs:");
    foreach (var element in material.InputSignature.Elements)
    {
        Console.WriteLine($"  {element.Name} ({element.Semantic})");
    }
}
```

## Example: Loading material textures

```csharp
var material = (Material)resource.DataBlock;
var fileLoader = new FileLoader();

foreach (var (paramName, texturePath) in material.TextureParams)
{
    if (!string.IsNullOrEmpty(texturePath))
    {
        using var textureResource = fileLoader.LoadFileCompiled(texturePath);
        if (textureResource?.DataBlock is Texture texture)
        {
            var bitmap = texture.GenerateBitmap();
            Console.WriteLine($"{paramName}: {texture.ActualWidth}x{texture.ActualHeight}");
        }
    }
}
```
