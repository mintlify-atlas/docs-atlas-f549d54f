## Overview

The `BinaryKV3` class represents a binary KeyValues3 data block. It provides methods for reading and deserializing binary KV3 format data into a structured object model.

**Namespace:** `ValveResourceFormat.ResourceTypes`

## Accessing from Resource

BinaryKV3 can be accessed from various blocks in a resource:

```csharp
using var resource = new Resource();
resource.Read("path/to/file_c");

// Access from DATA block
if (resource.DataBlock is BinaryKV3 data)
{
    var rootObject = data.Data;
    // Access properties
    var property = rootObject.GetProperty<string>("PropertyName");
}

// Access from specific block type
var ctrl = resource.GetBlockByType(BlockType.CTRL) as BinaryKV3;
if (ctrl != null)
{
    var ctrlData = ctrl.Data;
    // Process control block data
}
```

## Example: Reading BinaryKV3 properties

```csharp
var binaryKV3 = (BinaryKV3)resource.DataBlock;
var data = binaryKV3.Data;

Console.WriteLine($"Format: {binaryKV3.Format}");

// Read string property
if (data.ContainsKey("m_name"))
{
    var name = data.GetProperty<string>("m_name");
    Console.WriteLine($"Name: {name}");
}

// Read integer property
if (data.ContainsKey("m_nVersion"))
{
    var version = data.GetInt32Property("m_nVersion");
    Console.WriteLine($"Version: {version}");
}

// Read array
if (data.ContainsKey("m_Items"))
{
    var items = data.GetArray("m_Items");
    Console.WriteLine($"Items: {items.Length}");
    
    foreach (var item in items)
    {
        var itemName = item.GetProperty<string>("m_name");
        Console.WriteLine($"  - {itemName}");
    }
}
```

## Example: Converting to text format

```csharp
var binaryKV3 = (BinaryKV3)resource.DataBlock;

// Get as KV3File for text output
var kv3File = binaryKV3.GetKV3File();

// Write to console
using var writer = new IndentedTextWriter(Console.Out);
kv3File.WriteText(writer);

// Or convert to string
var textRepresentation = kv3File.ToString();
File.WriteAllText("output.kv3", textRepresentation);
```

## Example: Checking KV3 format

```csharp
using var stream = File.OpenRead("file.bin");
using var reader = new BinaryReader(stream);

var magic = reader.ReadUInt32();

if (BinaryKV3.IsBinaryKV3(magic))
{
    Console.WriteLine("This is a binary KV3 file");
    
    // Reset position and read
    stream.Position = 0;
    
    var binaryKV3 = new BinaryKV3();
    binaryKV3.Read(reader);
    
    Console.WriteLine($"Format: {binaryKV3.Format}");
    Console.WriteLine($"Root properties: {binaryKV3.Data.Properties.Count()}");
}
else
{
    Console.WriteLine("Not a binary KV3 file");
}
```

## Example: Accessing nested data

```csharp
var binaryKV3 = (BinaryKV3)resource.DataBlock;
var data = binaryKV3.Data;

// Access nested objects
if (data.ContainsKey("m_settings"))
{
    var settings = data.GetSubCollection("m_settings");
    
    if (settings.ContainsKey("m_quality"))
    {
        var quality = settings.GetProperty<string>("m_quality");
        Console.WriteLine($"Quality: {quality}");
    }
    
    // Access nested array
    if (settings.ContainsKey("m_options"))
    {
        var options = settings.GetArray("m_options");
        foreach (var option in options)
        {
            var key = option.GetProperty<string>("m_key");
            var value = option.GetProperty<string>("m_value");
            Console.WriteLine($"  {key} = {value}");
        }
    }
}
```

## Supported Versions

BinaryKV3 supports versions 1 through 5 of the binary KV3 format. The class automatically detects the version and applies the appropriate deserialization logic.

## Compression Support

BinaryKV3 supports multiple compression methods:
- **Uncompressed** (method 0)
- **LZ4** (method 1)
- **ZSTD** (method 2)

Compression is automatically detected and decompressed during the read operation.
