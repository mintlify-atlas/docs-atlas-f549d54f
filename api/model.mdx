## Overview

The `Model` class represents a model resource containing meshes, skeleton, animations, and related data. It provides methods for accessing embedded and referenced meshes, animations, physics data, and material groups.

**Namespace:** `ValveResourceFormat.ResourceTypes`

## Accessing from Resource

Model data is accessed from the DATA block of a resource:

```csharp
using var resource = new Resource();
resource.Read("path/to/model.vmdl_c");

var model = (Model)resource.DataBlock;

// Get model name
var name = model.Name;

// Get skeleton
var skeleton = model.Skeleton;
Console.WriteLine($"Bones: {skeleton.Bones.Length}");

// Get embedded meshes
foreach (var (mesh, meshIndex, meshName) in model.GetEmbeddedMeshes())
{
    Console.WriteLine($"Mesh {meshIndex}: {meshName}");
}
```

## Example: Loading model with meshes

```csharp
var model = (Model)resource.DataBlock;
var fileLoader = new FileLoader();

// Get embedded meshes
foreach (var (mesh, meshIndex, meshName) in model.GetEmbeddedMeshes())
{
    Console.WriteLine($"Embedded Mesh: {meshName}");
    
    // Get bone remap table for this mesh
    var remapTable = model.GetRemapTable(meshIndex);
    if (remapTable != null)
    {
        Console.WriteLine($"  Bone remap table has {remapTable.Length} entries");
    }
}

// Get referenced meshes
foreach (var (meshIndex, meshName, lodMask) in model.GetReferenceMeshNamesAndLoD())
{
    Console.WriteLine($"Reference Mesh: {meshName} (LoD: {lodMask})");
    
    using var meshResource = fileLoader.LoadFileCompiled(meshName);
    if (meshResource?.DataBlock is Mesh mesh)
    {
        mesh.GetBounds();
        Console.WriteLine($"  Bounds: {mesh.MinBounds} to {mesh.MaxBounds}");
    }
}
```

## Example: Loading animations

```csharp
var model = (Model)resource.DataBlock;
var fileLoader = new FileLoader();

// Get all animations (embedded + referenced)
var animations = model.GetAllAnimations(fileLoader);

foreach (var animation in animations)
{
    Console.WriteLine($"Animation: {animation.Name}");
    Console.WriteLine($"  Duration: {animation.Duration} seconds");
    Console.WriteLine($"  FPS: {animation.Fps}");
    Console.WriteLine($"  Frame count: {animation.FrameCount}");
}

// Get faceposer folders for organizing animations
var folders = model.GetFaceposerFolders();
foreach (var (animName, folderName) in folders)
{
    Console.WriteLine($"Animation '{animName}' in folder '{folderName}'");
}
```

## Example: Material and mesh groups

```csharp
var model = (Model)resource.DataBlock;

// Get mesh groups
var meshGroups = model.GetMeshGroups().ToList();
Console.WriteLine($"Mesh Groups: {string.Join(", ", meshGroups)}");

// Get default mesh groups
var defaultGroups = model.GetDefaultMeshGroups().ToList();
Console.WriteLine($"Default Groups: {string.Join(", ", defaultGroups)}");

// Get material groups
foreach (var (groupName, materials) in model.GetMaterialGroups())
{
    Console.WriteLine($"Material Group: {groupName}");
    foreach (var material in materials)
    {
        Console.WriteLine($"  - {material}");
    }
}
```
