This guide walks you through parsing a Source 2 resource file and extracting data using ValveResourceFormat.

## Prerequisites

Before starting, ensure you have:

- Installed ValveResourceFormat ([see installation guide](/installation))
- A Source 2 resource file to parse (files ending with `_c` like `.vtex_c`, `.vmdl_c`, `.vmat_c`)

<Note>
You can find Source 2 resource files in games like Dota 2, Half-Life: Alyx, Counter-Strike 2, or Deadlock. Files are typically located in VPK archives in the game's installation directory.
</Note>

## Basic resource parsing

The `Resource` class is the entry point for parsing any Source 2 resource file:

```csharp
using ValveResourceFormat;

// Open and read a resource file
using var resource = new Resource();
resource.Read("path/to/file.vtex_c");

// Access basic information
Console.WriteLine($"Resource type: {resource.ResourceType}");
Console.WriteLine($"File size: {resource.FileSize} bytes");
Console.WriteLine($"Version: {resource.Version}");
Console.WriteLine($"Block count: {resource.Blocks.Count}");
```

<Steps>
  <Step title="Create a Resource instance">
    The `Resource` class implements `IDisposable`, so use it with a `using` statement to ensure proper cleanup:
    
    ```csharp
    using var resource = new Resource();
    ```
  </Step>
  
  <Step title="Read the file">
    Call the `Read()` method with the file path:
    
    ```csharp
    resource.Read("textures/debug.vtex_c");
    ```
    
    The library will automatically detect the resource type based on the file extension.
  </Step>
  
  <Step title="Access the data">
    Once read, you can access blocks and data on the `resource` object:
    
    ```csharp
    var dataBlock = resource.DataBlock;
    var resourceType = resource.ResourceType;
    ```
  </Step>
</Steps>

## Reading from a stream

You can also read from a `Stream` instead of a file path:

```csharp
using var fileStream = File.OpenRead("path/to/file.vtex_c");
using var resource = new Resource()
{
    FileName = "path/to/file.vtex_c" // Optional but recommended
};
resource.Read(fileStream);

// Access the data
var texture = (Texture?)resource.DataBlock;
```

<Note>
Setting `FileName` helps the library determine the resource type if it can't be inferred from the stream.
</Note>

## Working with blocks

Source 2 resources are organized into blocks. Here's how to access them:

```csharp
using ValveResourceFormat;
using ValveResourceFormat.Blocks;

using var resource = new Resource();
resource.Read("model.vmdl_c");

// Get the DATA block
var dataBlock = resource.DataBlock;

// Get external references
var externalRefs = resource.ExternalReferences;

// Get a specific block by type
var vbibBlock = resource.GetBlockByType(BlockType.VBIB);

// Check if a block exists
if (resource.ContainsBlockType(BlockType.RERL))
{
    Console.WriteLine("Resource has external references");
}

// Iterate through all blocks
foreach (var block in resource.Blocks)
{
    Console.WriteLine($"Block type: {block.Type}, Size: {block.Size}");
}
```

## Example: Extracting a texture

Here's a complete example of extracting a texture as PNG:

<CodeGroup>
```csharp Simple extraction
using ValveResourceFormat;
using ValveResourceFormat.ResourceTypes;
using ValveResourceFormat.IO;

using var resource = new Resource();
resource.Read("textures/debug.vtex_c");

// Cast the DataBlock to Texture
var texture = (Texture)resource.DataBlock;

// Generate a bitmap
using var bitmap = texture.GenerateBitmap();

// Convert to PNG
var pngBytes = TextureExtract.ToPngImage(bitmap);

// Save to file
File.WriteAllBytes("output.png", pngBytes);
```

```csharp With error handling
using ValveResourceFormat;
using ValveResourceFormat.ResourceTypes;
using ValveResourceFormat.IO;

try
{
    using var resource = new Resource();
    resource.Read("textures/debug.vtex_c");
    
    if (resource.ResourceType != ResourceType.Texture)
    {
        Console.WriteLine($"Expected texture, got {resource.ResourceType}");
        return;
    }
    
    var texture = (Texture)resource.DataBlock;
    using var bitmap = texture.GenerateBitmap();
    var pngBytes = TextureExtract.ToPngImage(bitmap);
    
    File.WriteAllBytes("output.png", pngBytes);
    Console.WriteLine("Texture extracted successfully!");
}
catch (Exception ex)
{
    Console.WriteLine($"Error: {ex.Message}");
}
```
</CodeGroup>

## Example: Reading from VPK archives

Source 2 resources are often stored in VPK archive files. Use the [ValvePak library](https://github.com/ValveResourceFormat/ValvePak) to read them:

```csharp
using SteamDatabase.ValvePak;
using ValveResourceFormat;
using ValveResourceFormat.ResourceTypes;
using ValveResourceFormat.IO;

// Open the VPK package
using var package = new Package();
package.Read("pak01_dir.vpk");

// Find a file in the package
var entry = package.FindEntry("textures/debug.vtex_c");
if (entry == null)
{
    Console.WriteLine("File not found in package");
    return;
}

// Read the file data
package.ReadEntry(entry, out var fileData);

// Parse as a resource
using var memoryStream = new MemoryStream(fileData);
using var resource = new Resource()
{
    FileName = entry.GetFullPath()
};
resource.Read(memoryStream);

// Extract the texture
var texture = (Texture)resource.DataBlock;
using var bitmap = texture.GenerateBitmap();
var pngBytes = TextureExtract.ToPngImage(bitmap);

File.WriteAllBytes("output.png", pngBytes);
Console.WriteLine($"Extracted {entry.GetFullPath()}");
```

## Example: Working with entity lumps

Entity lumps contain map entity data:

```csharp
using ValveResourceFormat;
using ValveResourceFormat.ResourceTypes;

using var resource = new Resource()
{
    FileName = "maps/example.vents_c"
};
resource.Read("maps/example.vents_c");

var entityLump = (EntityLump)resource.DataBlock;

// Get all entities
var entities = entityLump.GetEntities();

foreach (var entity in entities)
{
    // Get the classname property
    var classname = entity.GetProperty<string>("classname");
    Console.WriteLine($"Entity: {classname}");
    
    // Access properties
    var properties = entity.Properties.Properties;
    Console.WriteLine($"  Properties: {properties.Count}");
}
```

## Using FileExtract helper

The `FileExtract` utility automatically handles extraction for various resource types:

```csharp
using ValveResourceFormat;
using ValveResourceFormat.IO;

using var resource = new Resource();
resource.Read("materials/example.vmat_c");

// Extract with automatic format detection
using var contentFile = FileExtract.Extract(resource, null);

// Write the main file
if (contentFile.Data != null)
{
    File.WriteAllBytes(contentFile.FileName, contentFile.Data);
}

// Write sub-files (e.g., referenced textures)
foreach (var subFile in contentFile.SubFiles)
{
    var subFileData = subFile.Extract();
    var outputPath = Path.Combine(
        Path.GetDirectoryName(contentFile.FileName) ?? "",
        subFile.FileName
    );
    
    Directory.CreateDirectory(Path.GetDirectoryName(outputPath) ?? "");
    File.WriteAllBytes(outputPath, subFileData);
}
```

## Error handling

Always handle potential errors when parsing resources:

```csharp
using ValveResourceFormat;

try
{
    using var resource = new Resource();
    resource.Read("file.vtex_c");
    
    // Process the resource
}
catch (InvalidDataException ex)
{
    Console.WriteLine($"Invalid resource file: {ex.Message}");
}
catch (UnexpectedMagicException ex)
{
    Console.WriteLine($"Unexpected file format: {ex.Message}");
}
catch (FileNotFoundException ex)
{
    Console.WriteLine($"File not found: {ex.Message}");
}
catch (Exception ex)
{
    Console.WriteLine($"Unexpected error: {ex.Message}");
}
```

<Warning>
Source 2 file formats may change between game updates. Always use the latest version of ValveResourceFormat to support newer formats.
</Warning>
